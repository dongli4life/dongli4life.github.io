詹宏志《读书与旅行》

Nov 18, 2018

【读书】

最初吸引我的是书名，毕竟这是我乐此不疲的两件事。巧的是大半本书恰好是在旅途中读完，带着“人在旅途”的心情来读这本书，在情感上便与作者有了共鸣，作者因读书而引发的旅程、跟随书本按图索骥的旅行方式，有读书人特有的某种“冥顽不灵”，于我而言是亲切可爱的。但客观地说，作品内容实与书名不符，大量的篇幅重在写美食游记，不如“旅行与美食”来得令人信服。

丹·布朗《本源》

Sep 23, 2018

【读书】

如今读丹·布朗的小说，情感上的满足远胜于作品内容本身，就小说而言，近几年的几部作品皆无法与《达芬奇密码》、《天使与魔鬼》相提并论，无论是构思还是语言的组织，似有读电影脚本的错觉。本书的人物对话中出现大段科学理论解释的文字，不精通此领域的读者实无法与之产生共鸣，徒留尴尬。但不得不说，若把这一系列小说集成《兰登教授历险记》系列电影，应该会很卖座吧，毕竟像当年的《哈利波特》一样，这些作品也见证了多少人的青春记忆。

克莱儿·麦克福尔《摆渡人2》

Sep 22, 2018

【读书】

不知是否翻译的缘故，语言略显稚嫩粗糙，虽为小说并不在文学性上有很高期待，但不经锤炼的文字还是令人读来失望。在情节上，第一部的惊喜消失殆尽，取而代之的是一切皆在预料之中的平淡。至于新增的两位主角苏珊娜和杰克，作者意图借此表达人们在不经意间相互影响，但实际读来似乎此种意味并不明确。

蔡崇达《皮囊》

Jul 24, 2018

【读书】

或许是多年记者的经历，哪怕是浓烈的感情都能通过冷静和理性的文字表达出来，这样的文笔很难让人不在情感共鸣的同时发生思考。就内容而言，胜在写长者的故事，有长者的爱情，有对神明的敬畏，有对故乡的感怀，一个个形象鲜明地跃然纸上；但作者疏于写同辈人的境遇，几乎是相似的人生轨迹——年少时的光彩和之后的落寞，而作者作为旁观者的角度却带着些许居高临下的意味，与前几篇中作者构建起的自身形象太过冲突，读来略有不适，有些失望。

毕淑敏《非洲三万里》

Jul 21, 2018

【读书】

如果作为对于非洲的历史文化、地理环境的科普读物，结合作者的游历所感，颇有趣味。然而，若因为书名的大气而对作品的文学性抱有期待，怕是会大感失望。另外，书中插图的编辑实在没有道理，读到作者笔下磅礴大气之景或新奇有趣之物，下意识看向书中配图却总是无迹可寻，可书中存在的插图实与文字并不相关或因为缺少说明而让人读来不明所以。不过，应该很多人读完后都会记住“非洲之傲”、若干个景点和埃塞俄比亚咖啡，从游记的角度来看，还是成功的。

林少华《小孤独》

Jul 12, 2018

【读书】

几乎不读日本文学作品，也就自然不读村上春树。《小孤独》吸引我的是书名和封面上对于“孤独”的注解，并不识得作者是村上作品的译者。开篇第一章写村上和其文学，笔下那个沉默寡言的老人与我印象里的村上全然不同，但又生动几分。一生翻译村上作品，当是知音了吧，这样的文字作者写来精诚所至，读来也是相当有说服力的。中间两章，感觉是一个刚退休尚有余热的老人在唠家常中回忆往昔，纵有书香气也掩映于“柴米油盐”中。最后一章所选文章读来并不明白与章节名“乡愁，诗和远方”有什么关系，读到“乡愁”，很难读到“诗和远方”。译者与其所译作品，很难说到底是谁影响谁更多一些，但联系总是有的。所以大概是因为我不读村上的关系，也就读不懂林少华吧。

江一燕《我是爬行者小江》

【读书】

Mar 15, 2018

【读书】

Mar 4, 2018

【读书】

Feb 1, 2018

【读书】

林语堂-苏东坡传

Oct 29, 2017

【读书】

喜欢苏轼的人读来一定不会失望，林语堂笔下天才的文学家，从政为民的好官，他的博学和惜才、通达和乐观，都使其成为一个近乎完美的典型，而其不得志的仕途，颠沛流离的一生，徒增后人对其的惋惜。但文人作传，总免不了主观情绪，这就造成了一定程度上对王安石的过度苛责。王安石所推崇的变法，其原意也是造福于民，但过于理想化和简单化使其实施效果产生诸多弊端，又因小人谄媚致使未能及时上传民意，这是历来身居高位者所面临的情形，并非王安石一人之过。不管怎样，小说笔法的传记通俗易懂，虽是一家之言，但终还是一部值得一读的作品。
