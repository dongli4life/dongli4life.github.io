弗兰克·迈考特《纽约，我来了！》

Feb 23, 2017

【读书】

南海出版社2010年版的书名为《安琪拉的灰烬2:就是这儿》。本书是《安琪拉的灰烬》的第二部，作者继续自传式文体，书写长大后孤身一人重返纽约，辗转多地从事着不同职业的奋斗历程。生活中与作者相伴的底层小人物，遍尝命运带来的诸多不幸却依旧关怀信仰的兴衰、下一代的前景和自我的价值实现。作者最终修得学业，成为教师，也收获自己的婚姻与家庭。与第一部相比，事件描述上不再一一事无巨细，但细节刻画依旧入木三分，尤其作者写初为教师的几年，让同为教师的我亦回想起当年，颇有共鸣也颇为感慨。
