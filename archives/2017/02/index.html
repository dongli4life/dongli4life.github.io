弗兰克·迈考特《纽约，我来了！》

Feb 23, 2017

【读书】

南海出版社2010年版的书名为《安琪拉的灰烬2:就是这儿》。本书是《安琪拉的灰烬》的第二部，作者继续自传式文体，书写长大后孤身一人重返纽约，辗转多地从事着不同职业的奋斗历程。生活中与作者相伴的底层小人物，遍尝命运带来的诸多不幸却依旧关怀信仰的兴衰、下一代的前景和自我的价值实现。作者最终修得学业，成为教师，也收获自己的婚姻与家庭。与第一部相比，事件描述上不再一一事无巨细，但细节刻画依旧入木三分，尤其作者写初为教师的几年，让同为教师的我亦回想起当年，颇有共鸣也颇为感慨。

太宰治《人间失格》

Feb 8, 2017

【读书】

自我剖析到极致的内心独白，将人性的弱点与黑暗面暴露无疑，但同时也显露出不健全的人格。每次读日本作家的小说，无论其题材与内容，读来皆有相似的压抑感，这些作品或写出了人敢想却不敢言的部分，因而常有人借以寻求某种心理慰藉，然而作品本身带有太过强烈的虚无感以致于整书皆是浓得化不开的沉重阴郁，这大致是我素来不喜欢读此类作品的原因。

弗兰克·迈考特《安琪拉的灰烬》

Feb 6, 2017

【读书】

这是一部自传小说，描述作者童年时举家从美国返回爱尔兰，酒鬼父亲无力担负起家庭之责，母亲却不断地孕育生命，居无定所、依靠救济金过活的一家人经历着贫穷潦倒、疾病侵扰、子女夭折，每日为吃穿用度的愁苦成为故事中最平常的事情。但就在这样一个社会最底层的街区，有毒舌却依然眷顾亲情的姊妹，有同为苦难者却乐观生活的邻居和工人，有古板严厉却鼓舞孩童的老师，所以纵使尝尽人间疾苦，作者仍能用平和的文字书写这段记忆，读来苦中带笑，绝境之处又分明看到希望。

马克·李维《生命里最美好的春天》

Feb 2, 2017

【读书】

这是湖南文艺出版社的版本，法文原版名为《自由的孩子》，上海译文出版社的版本直译其原版的书名，更为贴切。读过不少马克·李维的作品，作为一个畅销书作家，取材于二战背景，书写残酷战争中为自由而战的年少抵抗者，确与其他作品不用。宏大的题材，然而作者却延续其细腻传神的文字，虽悲壮豪迈之势弱了几分，但细微之处的痛惜与哀伤依旧打动人。
