张新颖《沈从文九讲》

Aug 31, 2017

【读书】

2014年读了《沈从文的后半生》，喜欢上作者的文字。《沈从文九讲》实则原本是大学授课的讲稿，用沈从文创作的时间顺序，结合具体作品梳理沈从文的文学思想的流变，书写从文学创作到历史文物研究背后那段鲜为人知的往事，用作品本身说话，读来有理有据。读张新颖的著作，才重识了沈从文。

唐诺《世间的名字》

Aug 22, 2017

【读书】

唐诺一直是我认为堪称“好读者”的典范之一，读过他的书评，总是旁征博引、细腻委婉地阐述观点，很有文人气。本书是他的散文作品，第一次读，文笔虽与其书评相似，但明显更肆意地将长句连缀成文，如不完全静下心来确实难以进入书中的世界，冗长又琐碎的表达初读来很不适应。不过丰富的阅历和知识储备读来的满足感，是唐诺的书一如既往的风格。

汪曾祺《人间有戏》

Aug 2, 2017

【读书】

儿时家庭中受父亲的影响耳濡目染与戏曲结缘，在机缘巧合之下，调回北京于北京京剧院任编剧数十年。汪老记录数十年间对于京剧剧本创作中的思考，回忆京剧伶人的艺术人生，讲述在特殊历史时期的相关经历，人生如戏，世间悲欢皆从容而谈。虽不了解梨园之事、戏曲之韵，但读汪老的文字还是从睿智机警处见到潇洒温暖。

汪曾祺《人间滋味》

Aug 1, 2017

【读书】

平淡轻松的文字一如既往是汪老的风格。书中记录天南地北“舌尖上的中国”，也借以抒怀感悟。写故乡高邮及江浙一带的特产，写西南联大时期学者们的食之经历，写人在旅途中偶遇的美食，汪老一生遍尝“人间滋味”，也在风土人情、个人经历中体悟“人间滋味”。文笔极富感染力，晚间阅读有消食之功效，然编辑尚待精进，文字雷同处太多，不过因其是“滋味”之集合，重复难免，尚可接受。
