林语堂-苏东坡传

Oct 29, 2017

【读书】

喜欢苏轼的人读来一定不会失望，林语堂笔下天才的文学家，从政为民的好官，他的博学和惜才、通达和乐观，都使其成为一个近乎完美的典型，而其不得志的仕途，颠沛流离的一生，徒增后人对其的惋惜。但文人作传，总免不了主观情绪，这就造成了一定程度上对王安石的过度苛责。王安石所推崇的变法，其原意也是造福于民，但过于理想化和简单化使其实施效果产生诸多弊端，又因小人谄媚致使未能及时上传民意，这是历来身居高位者所面临的情形，并非王安石一人之过。不管怎样，小说笔法的传记通俗易懂，虽是一家之言，但终还是一部值得一读的作品。

刘慈欣《三体》

Oct 15, 2017

【读书】

科幻故事融合于虚拟现实的游戏里，又结合到历史长河的现实中，的确让人有耳目一新的感觉。大段的数学的、物理的公式和理论，为小说增添了神秘感。在人物的设定上，读者很难把一个命运多舛的柔弱母亲的形象与终极破坏者联系起来，但其余人物间的关系书写得较为简单。从文学性的角度来看，故事说得不如故事内容本身的精彩。

阿兰·德波顿《写给无神论者》

Oct 7, 2017

【读书】

当初买这本书是因为作者的观点与我一直以来对于宗教的观点很一致，我不是任何一个宗教的信奉者和追随者，然而，每次行走到一处宗教场所，也总能感受到庄严肃穆的气氛所带来的敬畏之情。无论是世间的哪一个宗教，它的产生、教义的传播，多少有其时代的意义，凝聚着先人的智慧。时代的变迁，人类对于曾经未知的自然现象有了科学的解释，宗教沟通天人的作用逐渐被削弱，但其抚慰心灵之处依旧或多或少地影响着现代人。原著的书名更贴切地表达了作者的写作意图——Religion for Atheists: A Non-believer’s Guide to the Uses of Religion.

阿兰·德波顿《旅行的艺术》

Sep 17, 2017

【读书】

说旅行，其实着重说旅程中旅行者的内心世界，到达每一个目的地，联系与之相关的艺术家及其创作，展现他们对于旅行的不同观点及从中反映出的生命态度。作者的个人见解虽并无新颖之处，但每每读德波顿的书，总时有豁然开朗之感。就如同作者在《哲学的慰藉》中所表述的，他所写下的文字描述出我心中长久以来所想却未能用语言所描述出来的内容。

唐诺《世间的名字》

Aug 22, 2017

【读书】

唐诺一直是我认为堪称“好读者”的典范之一，读过他的书评，总是旁征博引、细腻委婉地阐述观点，很有文人气。本书是他的散文作品，第一次读，文笔虽与其书评相似，但明显更肆意地将长句连缀成文，如不完全静下心来确实难以进入书中的世界，冗长又琐碎的表达初读来很不适应。不过丰富的阅历和知识储备读来的满足感，是唐诺的书一如既往的风格。

汪曾祺《人间有戏》

Aug 2, 2017

【读书】

儿时家庭中受父亲的影响耳濡目染与戏曲结缘，在机缘巧合之下，调回北京于北京京剧院任编剧数十年。汪老记录数十年间对于京剧剧本创作中的思考，回忆京剧伶人的艺术人生，讲述在特殊历史时期的相关经历，人生如戏，世间悲欢皆从容而谈。虽不了解梨园之事、戏曲之韵，但读汪老的文字还是从睿智机警处见到潇洒温暖。

汪曾祺《人间滋味》

Aug 1, 2017

【读书】

平淡轻松的文字一如既往是汪老的风格。书中记录天南地北“舌尖上的中国”，也借以抒怀感悟。写故乡高邮及江浙一带的特产，写西南联大时期学者们的食之经历，写人在旅途中偶遇的美食，汪老一生遍尝“人间滋味”，也在风土人情、个人经历中体悟“人间滋味”。文笔极富感染力，晚间阅读有消食之功效，然编辑尚待精进，文字雷同处太多，不过因其是“滋味”之集合，重复难免，尚可接受。

加西亚·马尔克斯《霍乱时期的爱情》

Jul 28, 2017

【读书】

描述一份经历五十多年的爱情，在主人公与众多情人间的感情纠葛中变得较难有说服力，但结合当时的时代背景和作者的创作背景，可大致理解作者的创作意图。其父亲的爱情故事，在当时看来具有传奇色彩，加之马尔克斯的再加工，成就了小说的内容。除却人物情感关系本身，作者想借以表达的一种渴望爱与被爱却又不懂爱所产生的孤独感，正是推动小说情节发展、人物性格变化的主要原因。透过这一主题再反过来看待这部小说，方对其有新的认识。

J.R.R.托尔金《精灵宝钻》

Jul 12, 2017

【读书】

初读“创世录”和“维拉本纪”两个章节，人物谱系庞杂，很像希腊神话；“精灵宝钻征战史”此章节逐一叙述人物及各族群的故事，犹如魔戒版《旧约》。部分故事情节完整，与《魔戒》在记叙风格上一脉相承，其余则多似史学之作，无太多文学性。不过看过《魔戒》的人定要读一读这部作品，一来为满足对这部巨著的情结，二来为赞叹托尔金一人之力所创造出的如此大的神话体系。

潘富俊《草木缘情》

Jul 10, 2017

【读书】

想读这本书很久，初闻书名便无法忘记。草木历来都是文学作品中的常见之物，写草木以烘托环境、以记载四季时序、以抒怀言志。鲜少有人尝试真正将植物学与文学相联系，非精通于此二者之人不能为。作者作为一个植物学家，作品有科研著作的严谨，但相对缺少了文学性，论文气息尤甚，如将其归为植物科普之作，读来获益非浅，然归为文学之作，到底牵强。但作者跨越近半个世纪研究的精神，字里行间对于植物的亲切喜爱之情，还是让人钦佩。

格非《望春风》

Jul 9, 2017

【读书】

读此书之前，很是期待，写上个世纪五六十年代的乡村，写在时代洪流下的改革变迁，写各个年龄段的人不同的命运交织，这些主题是极易引发经历过那段岁月的读者共鸣的。书的前半部分写主人公离开乡村前所亲历的赵村故事，每个人物都栩栩如生，性格鲜明，书的后半部分写每个人物身上发生的命运转折，此时的文字略有生硬，人物的命运描绘略显苍白。此外，作者笔下的农村女性，或苍老，或卑微，或无助，或低贱，男性化的视角读来还是有待商榷。

徐则臣《王城如海》

Mar 4, 2017

【读书】

书名取自苏轼的诗“惟有王城最堪隐，万人如海一身藏”。书写不同社会阶层的人眼中的北京城，书中的人物皆不是在这座城市土生土长，从远离到回归，可以用省视的目光去看一座熟悉又陌生的城市，分明身处于此却又格格不入。从《耶路撒冷》到《王城如海》，大题目，却都书写小人物的内心，从乡村到城市的不适应，从海归到落地生根的不适应，欲望的城市背后充斥着孤寂的声音。整本小说不似当年读《耶路撒冷》那般打动我，倒是后记中作者描述写作这部小说期间所经历的无助与绝望让人动容。写小说的人总是不轻易将自己的柔弱示人，因为可以躲在人物背后让书中人物开口说话。因此在读后记的时候，竟一时分不清小说还是现实。

【读书】

全书共五个章节，写儿时记忆、民俗风物、西北大地、收藏摆设、自然景象。他写母亲，用朴实有温度的文字书写那个苦难时代的深刻记忆，让人唏嘘落泪。他写陕北的农民，写秦腔之于那片黄土地的意义，质朴的文字却有铿锵的力量。其他的随笔小品，读来皆是怀旧的文风，似有翻阅上世纪的文学作品之感。畅书心所欲言，是为“自在”；不羁于世俗评价，是为“独行”，然所书所感，依然是世间一俗人而已，但敢写最真实的自己，不由叫人钦佩。
